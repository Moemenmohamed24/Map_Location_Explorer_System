# Map_Location_Explorer_System

A location discovery system focused on Egypt that helps users find suitable places based on basic categories and structured data instead of generic map search results.

# Key Features (Current Version)
- Developed a structured database of popular places in Egypt.
- Implemented filtering system based on:
- Place type (recreational, cultural, historical, etc.)
- Built relational database to manage:
- Places data
- Categories and relationships
- Integrated external APIs for map-related data and visualization support.
- Designed and implemented using ASP.NET MVC architecture.

### Technical Highlights
- C# (ASP.NET MVC)
- SQL Server Database Design & Relationships
- REST API Integration
- Backend filtering and query logic
- Basic mapping integration using Google Maps JavaScript API

# Project Goal
The project addresses the limitation of traditional map services by organizing locations in a structured database that allows users to search and filter places more efficiently based on predefined categories. Future enhancements will include environmental and contextual filters such as air quality, weather, and sunlight levels.
